# kylemathias.github.io

Static portfolio site for `kylemathias.com`.

## Local setup

Requirements:

- Python 3.9+
- Node.js and npm are optional, only needed if you want the convenience scripts in `package.json`

Install steps:

1. Clone the repo.
2. From the repo root, start a local server with `npm run dev` or `python3 -m http.server 4321`.
3. Open `http://localhost:4321`.

## Project data generation

`projects.html` reads from `js/projects-data.js`, which is generated by `scripts/generate-projects-data.py`.

The generator expects a separate summaries directory:

- Default location: a sibling folder named `project-portfolio-summaries`
- Override with: `PROJECT_SUMMARY_DIR=/path/to/project-portfolio-summaries`

Example:

```bash
PROJECT_SUMMARY_DIR=../project-portfolio-summaries npm run generate:projects
```

## Notes

- `styles/main.css` is a committed stylesheet. There is no Tailwind build step in the current site.
- The `dist/` directory contains static output snapshots and is not required for local development.
